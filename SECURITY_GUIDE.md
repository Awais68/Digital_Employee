## 🚨 If Credentials Are Exposed

### Immediate Actions

1. **Revoke compromised password:**
   - Go to https://myaccount.google.com/apppasswords
   - Find and delete the exposed app password

2. **Generate new password:**
   - Create new app password in Google Account
   - Update `.env` file immediately

3. **Check for unauthorized access:**
   - Review Gmail activity
   - Check sent items for unknown emails

4. **Update Git history (if committed):**
   ```bash
   # If accidentally committed, remove from history
   git filter-branch --force --index-filter \
     "git rm --cached --ignore-unmatch .env" \
     --prune-empty --tag-name-filter cat -- --all
   
   # Force push (WARNING: rewrites history)
   git push origin --force --all
   ```
