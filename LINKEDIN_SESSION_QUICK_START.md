## 📝 Post to LinkedIn (No Login Needed!)

### Simple Post
```bash
python3 linkedin_mcp.py post "Your post content here #AI #Automation"
```

### Post from File
```bash
python3 linkedin_mcp.py post-file Approved/post.md
```

### Test Connection
```bash
python3 linkedin_mcp.py test
```
