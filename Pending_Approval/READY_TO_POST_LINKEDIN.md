1. **Get LinkedIn Access Token:**
   - Go to: https://www.linkedin.com/developers/tools/oauth/token-generator
   - Select scopes: `w_member_social`, `r_basicprofile`
   - Generate token
   - Update `.env` with new token

2. **Test Connection:**
   ```bash
   python3 linkedin_mcp.py test
   ```

3. **Publish Post:**
   ```bash
   # Move this file to Approved folder
   mv Pending_Approval/READY_TO_POST_LINKEDIN.md Approved/
   
   # Run orchestrator (will auto-publish)
   python3 orchestrator.py
   ```

4. **Verify:**
   - Check Logs/linkedin_log_*.md for post URL
   - Visit your LinkedIn profile to see the post
